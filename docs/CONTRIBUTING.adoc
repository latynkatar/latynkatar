= Парады па ўдзелу ў распрацоўцы


== Кодавыя пагадненні
Дакументацыя ствараецца ў фармаце https://asciidoc.org/[AsciiDoc]. 

Уся дакументацыя і каменты пішуцца па-беларуску. Назывы файлаў і аб'ектаў коду (зменных, канстантаў, класаў, метадаў,
функцый) мусяць быць зробленыя па-ангельску. 

Прымаецца даўжыня радка па 120 сімвалаў уключна.

У якасці фармату коду прымаецца той, да якога змушае https://github.com/psf/black[Black].
